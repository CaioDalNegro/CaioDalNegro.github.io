Some of my Work

Sistema de Gestão de Biblioteca

Java, CSV, POO

O Sistema de Gestão de Biblioteca foi desenvolvido para automatizar e organizar o processo de empréstimos e controle de livros, garantindo maior eficiência e redução de erros. Utilizando Java com Programação Orientada a Objetos (POO), a aplicação oferece funcionalidades como cadastro de livros, registro de empréstimos, devoluções, e consulta a históricos de usuários.

SGP - Sistema Gerenciador de Pousada

Python, Flask, HTML, CSS, JavaScript, TinyDB

O SGP - Sistema Gerenciador de Pousada foi desenvolvido como um projeto integrador no curso de Análise e Desenvolvimento de Sistemas da faculdade SENAI Antônio Adolpho Lobbe. Este projeto teve como objetivo solucionar problemas reais encontrados no site SAGA SENAI, baseando-se nos desafios enfrentados pela pousada Quinta do Ypuã.

Exemplos de Java JDBC

Java, MySQL e JavaJDBC

foi desenvolvido com o objetivo de demonstrar como utilizar a API JDBC (Java Database Connectivity) para interagir com bancos de dados relacionais de maneira prática.
